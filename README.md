# Learning Core

Обучающая система для изучения программирования с нуля через пошаговую практику.

---

## Цель проекта

Создать понятную систему обучения, которая позволяет человеку без опыта постепенно освоить основы программирования через практические действия и минимальную теорию.

---

## Принцип обучения

Обучение строится по простой логике:

1. Маленький шаг
2. Простое объяснение
3. Проверка понимания
4. Практическое действие
5. Проверка результата
6. Переход дальше

---

## Ключевая идея

Понимание формируется через действие, а не через запоминание.

---

## Уровни обучения

- Уровень 0 — мышление и логика
- Уровень 1 — базовый синтаксис
- Уровень 2 — работа с данными
- Уровень 3 — логика программ
- Уровень 4 — мини-проекты
- Уровень 5 — реальные проекты

---

## Философия системы

- Минимум теории
- Максимум практики
- Каждый шаг должен быть понятен без опыта

Главная цель — не выучить язык, а научиться мыслить как разработчик.
