# learning-core
Обучающая система для изучения программирования с нуля через пошаговое практическое освоение
## Цель проекта
Создать простую и понятную структуру обучения, которая позволяет человеку без опыта постепенно освоить основы программирования через практику и минимальную теорию.
## Принцип работы
Обучение строится по принципу:
1. Маленький шаг
2. Простое объяснение
33.проверка усвоеного тестированием
4. Практическое действие 
5. Проверка понимания и работоспособности 
6. Переход дальше 
## Структура обучения
 Уровень 0 — базовое понимание логики и мышления 
- Уровень 1 — основы кода 
- Уровень 2 — работа с данными
- Уровень 3 — простые приложения 
- Уровень 4 — проекты 
## Философия системы
Не перегружать теорией.  
Каждый шаг должен быть понятен без опыта.
Главная цель — не знание языка, а понимание логики программирования.
